# 🛒 E-Commerce Frontend

A modern, responsive and user-friendly **E-Commerce Frontend Web Application** built using **Vanilla JavaScript**. This project demonstrates my learning and practical implementation of core JavaScript concepts by building a real-world application.

## 🚀 Features
- 🛍️ Browse Products  
- 🔍 Search & Filter Products  
- 🛒 Add to Cart  
- ❌ Remove from Cart  
- 🧮 Real-time Cart Total Calculation  
- 📱 Responsive Design (Mobile + Desktop)

## 🧠 Learning & Implementation
This project is part of my journey to master JavaScript. While building this project, I focused on:
- Writing clean and structured JavaScript code  
- Understanding DOM Manipulation deeply
